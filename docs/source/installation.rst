Installation
============

Stable Release
--------------

Use `pip (Package manager for python) <https://pip.pypa.io/en/stable/>`_ to install the latest stable release::

   pip3 install verda

Install from source
-------------------

To install the package from the source files, first download them from the `GitHub repository <https://github.com/verda-cloud/sdk-python>`_::

    git clone https://github.com/verda-cloud/sdk-python.git

Then install it by::

    python3 setup.py install
